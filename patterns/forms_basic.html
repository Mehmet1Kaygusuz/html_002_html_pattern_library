Basic Form Structure

Forms use the form element to collect user input. The action attribute specifies where to send form data, and method specifies how to send it (GET or POST).

Simple Contact Form







<form action="#" method="POST">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name" required>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    
    <label for="message">Message:</label>
    <textarea id="message" name="message" rows="4" cols="30"></textarea>
    
    <button type="submit">Send Message</button>
</form>

Common Input Types

HTML provides various input types for different kinds of data.

Different Input Types













<form action="#" method="GET">
    <label for="text-input">Text:</label>
    <input type="text" id="text-input" name="text-input" placeholder="Enter text">
    
    <label for="password-input">Password:</label>
    <input type="password" id="password-input" name="password-input">
    
    <label for="email-input">Email:</label>
    <input type="email" id="email-input" name="email-input">
    
    <label for="number-input">Number:</label>
    <input type="number" id="number-input" name="number-input" min="1" max="100">
</form>

Checkboxes and Radio Buttons

Checkboxes allow multiple selections, while radio buttons allow only one selection from a group.

Checkboxes

Select your interests:





<input type="checkbox" id="html" name="interest" value="html">
<label for="html">HTML</label>

<input type="checkbox" id="css" name="interest" value="css">
<label for="css">CSS</label>

Radio Buttons

Select your experience level:





<input type="radio" id="beginner" name="level" value="beginner">
<label for="beginner">Beginner</label>

<input type="radio" id="intermediate" name="level" value="intermediate">
<label for="intermediate">Intermediate</label>

Select Dropdown

The select element creates a dropdown menu with multiple options.

Basic Dropdown



<label for="country">Choose your country:</label>
<select id="country" name="country">
    <option value="">--Please choose--</option>
    <option value="uk">United Kingdom</option>
    <option value="usa">United States</option>
    <option value="canada">Canada</option>
</select>

Common Form Attributes

HTML provides several attributes to control form behavior and validation.

Form with Various Attributes











<input type="text" id="username" name="username" required>

<input type="url" id="website" name="website" value="https://example.com" readonly>

<textarea id="comments" name="comments" disabled>This field is disabled</textarea>

<input type="number" id="age" name="age" min="18" max="100">

<input type="text" id="pattern-input" pattern="[A-Za-z]+" title="Only letters allowed">

<input type="hidden" name="user-id" value="12345">
