Basic Table Structure

A table consists of rows (tr) and cells. Cells can be data cells (td) or header cells (th).

Simple Data Table

Name Age City
Alice 25 London
Bob 30 Manchester
Charlie 28 Birmingham 
<table border="1">
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>City</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>25</td>
        <td>London</td>
    </tr>
    <tr>
        <td>Bob</td>
        <td>30</td>
        <td>Manchester</td>
    </tr>
</table>

Table with Semantic Sections

Tables can be divided into thead (header), tbody (body), and tfoot (footer) for better structure and accessibility.

Table with thead, tbody, and tfoot

Product Price Quantity
Laptop £800 2
Mouse £15 5
Keyboard £45 3
Total Items: 10 
<table border="1">
    <thead>
        <tr>
            <th>Product</th>
            <th>Price</th>
            <th>Quantity</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Laptop</td>
            <td>£800</td>
            <td>2</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td colspan="2">Total Items:</td>
            <td>10</td>
        </tr>
    </tfoot>
</table>

Colspan and Rowspan

Use colspan to merge cells horizontally and rowspan to merge cells vertically.

Colspan Example

Name Contact Information
Alice alice@email.com +44 7123 456789
Bob bob@email.com +44 7987 654321 
<table border="1">
    <tr>
        <th>Name</th>
        <th colspan="2">Contact Information</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>alice@email.com</td>
        <td>+44 7123 456789</td>
    </tr>
</table>

Rowspan Example

Day Morning Afternoon
Monday Foundation Mathematics Foundation Computing
Study Session Project Work
Tuesday Technology Group Project Free Time 
<table border="1">
    <tr>
        <th>Day</th>
        <th>Morning</th>
        <th>Afternoon</th>
    </tr>
    <tr>
        <td rowspan="2">Monday</td>
        <td>Foundation Mathematics</td>
        <td>Foundation Computing</td>
    </tr>
    <tr>
        <td>Study Session</td>
        <td>Project Work</td>
    </tr>
</table>

Complex Table Example

Combining colspan, rowspan, and semantic sections creates complex table structures.

Course Schedule Table

Time Week Days
Monday Wednesday Friday
9:00 - 11:00 Foundation Mathematics Foundation Computing Technology Group Project
13:00 - 15:00 Self-Study / Lab Work 
<table border="1">
    <thead>
        <tr>
            <th rowspan="2">Time</th>
            <th colspan="3">Week Days</th>
        </tr>
        <tr>
            <th>Monday</th>
            <th>Wednesday</th>
            <th>Friday</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>9:00 - 11:00</td>
            <td>Foundation Mathematics</td>
            <td>Foundation Computing</td>
            <td>Technology Group Project</td>
        </tr>
    </tbody>
</table>
