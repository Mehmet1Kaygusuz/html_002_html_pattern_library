What are HTML Entities?

HTML entities are special codes used to display reserved characters or symbols that would otherwise be interpreted as HTML code. They begin with an ampersand (&) and end with a semicolon (;).

Entities are essential when you want to display characters like <, >, &, or special symbols like ©, €, or  .

Common HTML Entities

Here are the most commonly used HTML entities:

Result Entity Code Description
  &nbsp; Non-breaking space
< &lt; Less than sign
> &gt; Greater than sign
& &amp; Ampersand
" &quot; Quotation mark

© &copy; Copyright symbol
® &reg; Registered trademark

€ &euro; Euro currency symbol
£ &pound; Pound sterling symbol
¥ &yen; Yen currency symbol

Displaying HTML Code

To display HTML code as text on a webpage, you must use entities to prevent the browser from interpreting it as actual HTML.

Example: Displaying HTML Tags

To show this code: <p>Hello World</p>

You must write: &lt;p&gt;Hello World&lt;/p&gt;

Example: Showing Code in Pre Tag

<div class="container">
    <h1>Page Title</h1>
    <p>Content goes here</p>
</div>

How HTML Handles Whitespace

HTML collapses multiple whitespace characters (spaces, tabs, line breaks) into a single space. This is called "whitespace collapsing."

Understanding this behavior is important for controlling how text appears on your webpage.

Multiple Spaces Example

No matter how many spaces you add in your HTML code, the browser will display only one space.

Code with Multiple Spaces

<p>This    text    has    many    spaces.</p>

Result (spaces collapsed)

This text has many spaces.

Using Non-Breaking Spaces

To preserve multiple spaces, use &nbsp; entity:

This    text    has    preserved    spaces.

<p>This&nbsp;&nbsp;&nbsp;&nbsp;text&nbsp;&nbsp;&nbsp;&nbsp;has&nbsp;&nbsp;&nbsp;&nbsp;preserved&nbsp;&nbsp;&nbsp;&nbsp;spaces.</p>

Line Breaks and Tabs in HTML

Line breaks and tabs in your HTML source code are also collapsed into single spaces unless you use specific elements.

Line Breaks in Code

<p>
This text
spans multiple
lines in the code
</p>

Result (line breaks collapsed)

This text spans multiple lines in the code

Using <br> for Line Breaks

lines using br tags

<p>
This text<br>
spans multiple<br>
lines using br tags<br>
</p>

The <pre> Element

The pre element preserves whitespace exactly as it appears in your code. Spaces, line breaks, and tabs are all maintained.

Text in <pre> Element

This    text    has    multiple    spaces.
It also spans
    multiple lines
        with indentation
            preserved.

Code Example

<pre>This    text    has    multiple    spaces.
It also spans
    multiple lines
        with indentation
            preserved.</pre>

Practical Examples

Here are some real-world scenarios where entities and whitespace handling matter:

Example 1: Copyright Footer

© 2026 Mehmet Kaygusuz. All rights reserved.

<p>&copy; 2026 Mehmet Kaygusuz. All rights reserved.</p>

Example 2: Price Display

Price: £49.99 | €55.00 | ¥6,800

<p>Price: &pound;49.99 | &euro;55.00 | &yen;6,800</p>

Example 3: Mathematical Expression

The formula is: x × y ÷ z ± 10

<p>The formula is: x &times; y &divide; z &plusmn; 10</p>

Example 4: Showing HTML Code

Use the <strong> tag for important text.

<p>Use the &lt;strong&gt; tag for important text.</p>
