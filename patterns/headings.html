Basic Heading Levels

HTML provides six levels of headings, from h1 to h6. These elements establish a clear content hierarchy and help both users and assistive technologies understand the structure of a page.

h1

<h1>h1</h1>

h2

<h2>h2</h2>

h3

<h3>h3</h3>

h4

<h4>h4</h4>
h5
<h5>h5</h5>
h6
<h6>h6</h6>

Blog Article Structure

This example demonstrates how heading levels can be used to structure a typical blog article, with a main title, primary sections, and nested subsections.

Blog Title (h1)

<h1>Blog Title (h1)</h1>

Main Section (h2)

<h2>Main Section (h2)</h2>

Subsection (h3)

<h3>Subsection (h3)</h3>

Detail Level (h4)

<h4>Detail Level (h4)</h4>
Minor Detail (h5)
<h5>Minor Detail (h5)</h5>
Smallest Detail (h6)
<h6>Smallest Detail (h6)</h6>
