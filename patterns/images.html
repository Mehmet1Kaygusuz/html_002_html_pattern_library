Basic Image Element

The img element embeds an image into the document. It requires a src attribute to specify the image source.

Simple Image with Alt Text

<img src="https://via.placeholder.com/400x300" alt="Placeholder image showing 400x300 dimensions">

The Importance of Alt Attributes

The alt attribute provides alternative text for images. It is crucial for accessibility and SEO.

Good Alt Text Example

<img src="..." alt="Blue rectangular button with white text">

Poor Alt Text Example (Avoid)

<img src="..." alt="image">

Avoid generic alt text like "image" or "picture". Describe what the image shows.

Image Width and Height

You can specify image dimensions using width and height attributes to prevent layout shifts while the page loads.

Image with Specified Dimensions

<img src="https://via.placeholder.com/600x400" 
     alt="Placeholder image with specified dimensions" 
     width="300" 
     height="200">

Common Image Formats

Different image formats serve different purposes. Here are the most common formats used on the web:

JPEG (.jpg, .jpeg)
Best for photographs and images with many colors. Supports compression but loses quality.
PNG (.png)
Supports transparency. Best for logos, icons, and images requiring sharp edges.
WebP (.webp)
Modern format with better compression than JPEG and PNG. Supports both lossy and lossless compression.
SVG (.svg)
Vector format that scales infinitely without quality loss. Best for logos and simple graphics.
GIF (.gif)
Supports animation but limited to 256 colors. Mostly used for simple animations.

Picture Element for Responsive Images

The picture element allows you to provide multiple image sources for different screen sizes or formats.

Basic Picture Element

<picture>
    <source media="(min-width: 800px)" srcset="https://via.placeholder.com/800x600">
    <source media="(min-width: 400px)" srcset="https://via.placeholder.com/400x300">
    <img src="https://via.placeholder.com/200x150" alt="Responsive placeholder image">
</picture>

Figure and Figcaption

Use the figure element to wrap images that need captions, providing semantic meaning.

Image with Caption

<figure>
    <img src="https://via.placeholder.com/500x300" alt="Example chart showing data visualization">
    <figcaption>Figure 1: Sample data visualization chart</figcaption>
</figure>
