List Patterns

Examples demonstrating unordered lists, ordered lists, and definition lists in HTML.

Back to index

Unordered Lists

Unordered lists are used when the order of items does not matter. Each item is marked with a bullet point by default.

Basic Unordered List

  • HTML
  • CSS
  • JavaScript
<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>

Nested Unordered List

  • Frontend
    • HTML
    • CSS
    • JavaScript
  • Backend
    • Node.js
    • Python
    • PHP
<ul>
    <li>Frontend
        <ul>
            <li>HTML</li>
            <li>CSS</li>
            <li>JavaScript</li>
        </ul>
    </li>
    <li>Backend
        <ul>
            <li>Node.js</li>
            <li>Python</li>
            <li>PHP</li>
        </ul>
    </li>
</ul>

Ordered Lists

Ordered lists are used when the sequence of items matters. Items are numbered automatically.

Basic Ordered List

  1. Open VS Code
  2. Create a new file
  3. Write HTML code
  4. Save the file
  5. Open in browser
<ol>
    <li>Open VS Code</li>
    <li>Create a new file</li>
    <li>Write HTML code</li>
    <li>Save the file</li>
    <li>Open in browser</li>
</ol>

Nested Ordered List

  1. Plan your project
    1. Define goals
    2. Research tools
    3. Create timeline
  2. Build your project
    1. Set up environment
    2. Write code
    3. Test features
  3. Deploy your project
<ol>
    <li>Plan your project
        <ol>
            <li>Define goals</li>
            <li>Research tools</li>
            <li>Create timeline</li>
        </ol>
    </li>
    <li>Build your project
        <ol>
            <li>Set up environment</li>
            <li>Write code</li>
            <li>Test features</li>
        </ol>
    </li>
    <li>Deploy your project</li>
</ol>

Definition Lists

Definition lists are used to display terms and their descriptions, often used for glossaries or metadata.

Basic Definition List

HTML
HyperText Markup Language - the standard markup language for web pages.
CSS
Cascading Style Sheets - used for styling HTML elements.
JavaScript
A programming language that enables interactive web pages.
<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language - the standard markup language for web pages.</dd>
    
    <dt>CSS</dt>
    <dd>Cascading Style Sheets - used for styling HTML elements.</dd>
    
    <dt>JavaScript</dt>
    <dd>A programming language that enables interactive web pages.</dd>
</dl>
